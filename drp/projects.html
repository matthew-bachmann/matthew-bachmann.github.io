Project List

This is a list of topics proposed by current graduate students interested in mentoring this semester.

Discrete Math

  • (Extremal Combinatorics) This project will look at topics in extremal combinatorics (graph and poset theory) such as intersecting set, shadow minimization, maximum weight ideal, and isoperimetric theorems, as well as relationships between these problems in the hypercube.
  • (Graph Theory) This project has a variety of options, such as: matchings, flows, Hamiltonian and Eulerian walks, colorings, shortest path algorithms, trees, planar graphs, and connectivity. A project exploring platonic graphs is also an option.
