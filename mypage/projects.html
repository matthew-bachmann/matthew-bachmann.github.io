Nueral Network

I'm currently teaching myself Python by trying to build a nueral network from scratch (I know there are libraries for this that are much more efficient than what I can write). I was inspired by watching Sebastian Lague's youtube video where he builds a nueral network to classify hand drawn images. That channel is great and I highly recommend checking it out! While the video explains ML concepts in a very accessible way, it does leave out some of the coding details. My goal is to recreate this project in Python and find ways to reproduce the beautiful graphs produced in the video. Here I will catalogue my journey, all code can be found on GitHub.
